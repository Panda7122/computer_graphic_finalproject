1.use left and right arrow key move the block
2.use Z and up arrow key rotate the block
3.use C to hold the block(you can use only one time before the block touch the ground)

5.use space key to let block drop on ground
6.use 'p' to pause the game
7.use 'r' to restart the game
8.use "wasd" to move your location
9.use '/','*' to flying up, down
10.use 'F' to switch third and first person
11.use mouse to Control your facing
